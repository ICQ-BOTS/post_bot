# Описание
Я создатель ботов для администрирования новых публикаций!

- База данных: sqlite3
- python3.6

# Установка

1. Установка всех зависимостей 
```bash
pip3 install -r requirements.txt
```

2. Инициализация базы данных
```bash
python3 database.py
```

3. Вставляем токен в config.py

4. Запуск бота!
```bash
python3 post_bot.py
```
